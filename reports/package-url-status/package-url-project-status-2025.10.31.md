## PURL Specification Update - October 31, 2025

### **Progress**
* Completed editorial changes for the PURL Ecma standard in response to feedback from the Ecma Secretariat
* Updated Package-URL website to streamline it for first version
   *  Updated PURL type grid with links to PURL type definitions (and added bazel and julia types)
   *  Created Tools grid (for PURL and VERS)
* Approved 2 new PURL types - Bazel and Julia.
* Closed 4 Issues and 28 Pull Requests for PURL (3 were not merged for various reasons)
* Closed 5 Pull Requests for VERS (1 was not merged)

### **Plans for November**
* Create v1.0 release of purl-spec repository
* Publish the Package-URL website
* Complete proposal to reset the PURL test schema and test suite to align with the v1.0 Ecma standard
* Create template(s) and update documentation for submitting requests to register a new PURL Type 
* Continue to work down the backlog of requests to register new PURL types

### **PURL outreach and in the News**
* PURL added to CVE Record Format v5.2.0 - see https://github.com/CVEProject/cve-schema/releases/tag/v5.2.0
* PURL added to primary display for packages at https://central.sonatype.com/
