## PURL Specification Update - August 31, 2025

### **Progress**

* Moved PURL v1.0 close to Ecma standard for submission to Ecma (delayed by a week)
* Completed restructuring the purl-spec repo   
  * Converted documentation files from rST to markdown format for easier contributor access and use in the new Package-URL website  
  * Moved documentation from the PURL-SPECIFICATION.rst and PURL-TYPES.rst files to more granular markdown files with a clear distinction between files that are for the Ecma standard and those that are for the broader specification  
* Closed 12 Issues and 20 Pull Requests (6 were not merged for various reasons)
* Completed first version of vers-spec/schemas/vers-test.schema.json

### **Plans for September**
* Submit PURL v1.0 Standard to TC54 and Ecma for review
* Create v1.0 release of purl-spec repository
* Create template(s) and update documentation for submitting requests to register a new PURL Type 
* Continue to work down the backlog of requests to register new PURL types
* Continue to work on improving and adding PURL type tests
