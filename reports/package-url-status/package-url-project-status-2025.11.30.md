## PURL Specification Update - November 30, 2025

### **Progress**
* Updated Package-URL website:
   *  Updated Tools grid (for PURL and VERS) with community feedback.
   *  Separated specifications from tools and researched data for both to complete the grid. 
   *  Focus is on adoption by other open source projects and other specifications.
* Completed proposal to reset the PURL test schema and test suite to align with the v1.0 Ecma standard.
* Closed 3 Issues and 7 Pull Requests for PURL ( 3 were not merged for various reasons).
* Closed 1 Pull Request for VERS.

### **Plans for December**
* Create v1.0 release of purl-spec repository.
* Reset documentation in purl-spec repo to match final Ecma text.
* Publish the Package-URL website.
* Create template(s) and update documentation for submitting requests to register a new PURL Type .
* Continue to work down the backlog of requests to register new PURL types.

### **PURL outreach and in the News**
* Packagist added PURL to package pages and their security advisories API - see Composer [Issue 1497](https://github.com/composer/packagist/issues/1497).
* AcquaSecurity's new [vexhub project](https://github.com/aquasecurity/vexhub) is based on PURL for software identification.
